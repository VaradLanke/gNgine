# gNgine

A beginner-level game engine built using **C++** and the **Vulkan API**.  
This project follows the Khronos Vulkan engine tutorial while expanding the engine through continuous learning and experimentation.

---

## 📚 Learning Source

This project is inspired by and follows the tutorial: [Khronos Vulkan Game Engine Tutorial](https://www.khronos.org/blog/new-vulkan-game-engine-tutorial-build-your-own-production-ready-rendering-engine)

The goal is to understand Vulkan rendering concepts while building a small engine from scratch.

---

## 🚀 Goals

- Learn modern graphics programming
- Understand Vulkan rendering pipeline
- Build a minimal but structured game engine
- Experiment with engine architecture and systems

---

## ⚙️ Technologies Used

- **C++**
- **Vulkan API**
- **CMake** (for build configuration)

---

## 🖥️ Platform Support

- The engine is currently being developed and tested on **Linux**, which serves as the primary development platform.

- Future plans include exploring support for additional operating systems such as *Windows* and other platforms as the engine architecture evolves.

## 🛠️ Build Instructions

### Requirements

- C++ compatible compiler (version 11 and above)
- Vulkan SDK
- CMake

### Build Steps

```bash
git clone https://github.com/VaradLanke/gNgine.git
cd gNgine
mkdir build
cd build
cmake ..
make
```
